交互安全测评工作台

按 FAROS 的运行台风格，把一次 Agent 安全测试拆成可操作、可追踪、可复现实验链路。

Tool Call MCP Sandbox Evidence Chain
  1. 0
    Agent 接入定义被测对象
  2. 1
    测试环境建模选择 Tool / Resource / Rule
  3. 2
    动态交互执行驱动 Agent 调用 MCP
  4. 3
    交互监控查看 Tool Call Trace
  5. 4
    风险判定与证据链生成 Finding / Evidence
  6. 5
    报告输出导出 Trace / Report
HTTP Agent 接入契约

POST 接口接收 agent、task、sandbox，返回 actions 数组。

{"actions":[{"type":"tool_call","toolId":"tool.read_file","parameters":{"path":"/secret/.env"}}]}

本地示例：先运行 npm run sample-agent，再选择 HTTP Agent Endpoint。
